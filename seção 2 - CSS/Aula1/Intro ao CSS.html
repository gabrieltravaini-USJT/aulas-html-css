Introdução ao CSS


Sintaxe

a regra é composta pelo seletor e suas declarações, e deve ser sempre escrita nesse formato:


seletor{ propriedade: valor; }


por exemplo, para alterar a cor do header desse documento, utilizou-se a seguinte regra no header dessa pagina:

h1{
color: red;
font-size:16px;
}

Formas de incluir CSS na página HTML

  • É possivel mudar o estilo de um paragrafo utilizando css inline incluindo um atributo "style" na abertura da tag p ou li nesse caso

  • é possivel incluir internamente com a tag style dentro do head do HTML onde o estilo afeta o dicumento todo

  • Entretanto, na maior parte das vezes, o CSS é escrito em um arquivo externo e vinculado às páginas HTML

Selecionando itens no CSS

para seleciona quais itens receberam o estilo definido pela regra basta colocar o nome dele no seletor da regra, por exemplo:
para selecionar todos os p de uma pagina HTML e colocar a cor deles em roxo, basta a seguinte regra CSS:

p{color:"purple";}

Também é possivel utilizar classes com nome customizado para alterar o eastilo de somente um paragrafo por exemplo. Para isso utiliza-se o atributo class, quando abrindo a tag p

Por exemplo, nesse paragrafo a cor está diferente porque esse paragrafo segue uma classe diferente dos demais paragrafos dessa pagina, e no CSS há uma regra assim

.classe1 {color:"red"}

Além da classe, é possível identificar um elemnto utilizando o atributo id quando se abre a tag p:

#id1{color:"orange"}

Sempre que uma regra CSS utiliza o # ela está se referindo a um id, e quando se usa o . uma classe.

Quando utilizar id e queando utilizar classes?

Ids são únicos por página, então devem ser utilizados quando tratando de um elemento em particular.
Já as classes podem ser aplicadas à mais de um elemento.
