Algumas propriedades do CSS

Font

para editar a fonte da seleção atual:

  • font-family - edita a familia da fonte
  • font-size - edita o tamanho da fonte
  • font-style - edita o estilo da fonte (itálico)
  • font-weight - edita o "peso" da fonte (negrito)
  • font-variant - especifica se existe alguma variante na fonte (maiusculo, minusculo etc.).
NÃO EXISTE FONT-COLOR


Text

Para editar o texto dentro da seleção:

  • text-align - para definir o alinhamento do texto.
  • text-decoration - para sublinhado, cortado, etc.
  • text-indent - para definir o espaçamento no inicio do paragrafo
  • text-transform - para escrever tudo maiúsculo opu minúsculo
NÃO EXISTE TEXT-COLOR


Cores

RGB

para definir uma cor em RGB utiliza-se a #, seguida de tres pares de numeros Hexadecimais, onde os dois primeiros representam o vermelho, os dois do meio, o verde e os dois ultimos o azul. exemplos a seguir:

  • #FFFFFF - é branco, pois é a soma das tres Cores.
  • #FF0000 - é o vermelho, pois os dois primeiros digitos responsáveis pelo vermelho estão completos enquanto o resto está zerado.
  • #00FF00 - é o verde.
  • #0000FF - é o azul.
