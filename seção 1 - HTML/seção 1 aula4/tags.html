  • ul inicia uma lista não ordenada

links

a para link - google nesse caso. Links precisam de atributos, nesse caso, o endereço do link.

sintaxe para atributos é sempre "tag atributo = 'valor'"


para abrir um link em outra aba, é necessário incluir um parametro target="_blank" no argumento a


tipos de links

  • href="https://www.google.com" para outros sites
  • href="outrapag.html" para outra pag dentro do mesmo site
  • href="#secao" para outra seção dentreo da mesma pag
  • href="mailto:alguem@server.com" para email, entretanto só funciona com outlook/firebird, etc.
